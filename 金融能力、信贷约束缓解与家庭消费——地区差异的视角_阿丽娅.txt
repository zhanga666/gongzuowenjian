# 金融能力、信贷约束缓解与家庭消费——地区差异的视角
阿丽娅 王汀汀 韩复龄


## 摘要
随着扩内需改革深化与消费环境改善，家庭消费升级研究成为学术界焦点。金融机构信贷支持可缓解家庭信贷约束以促进消费，该作用受家庭内部金融能力影响且存在地区差异。本文分析居民金融能力对家庭分类消费的影响及作用机制，实证结果表明：  
1. 居民金融能力对家庭总消费（尤其**质量型消费**）产生显著正向影响；  
2. 该促进效应存在明显城乡与地区差异：城镇地区金融能力对质量型消费的作用大于农村地区，西部地区大于东、中部地区；  
3. 缓解信贷约束后，家庭的**正规负债**、**财富**及社会资本中的**亲缘关系**，是金融能力促进总消费与质量型消费的核心解释因素。  

**关键词**：金融能力；信贷约束缓解；质量型消费  
**JEL 分类号**：D14；G19；R22


## 一、引言
新冠肺炎疫情冲击下，我国政府在“稳就业、保民生”基础上实施扩大内需战略，推动消费回升以转变经济发展方式。同时，国际经济环境复杂、出口不稳定、投资向房地产倾斜，消费成为经济增长核心动力，高质量消费孕育新增长潜力。  

家庭消费意愿与能力受**金融能力**显著影响：在复杂经济环境中，金融能力强（能制定良好金融决策、与金融机构有效互动）的个体，更易实现金融目标、改善家庭福利、对冲风险，进而支持经济增长。金融能力已成为高收入国家与新兴经济体政策制定者的优先考量（贝多广，2019）。  

本文的研究意义体现在三方面：  
- **家庭层面**：培养金融能力可优化消费行为，满足高品质生活需求；  
- **金融机构层面**：加强金融教育以提升居民金融能力，能提高金融工具使用效率与客户满意度；  
- **政府层面**：制定针对性地区金融支持与消费刺激政策，可更高效推动消费升级。  

国内外学者已探讨金融能力内涵（如世界银行2013年定义其包含金融知识、技能、态度、行为；贝多广2019年强调“金融知识+技能+生态+实践行为”），但现有研究仍存在不足：未充分覆盖金融能力的完整维度，且忽略我国城乡二元结构与地区差异对金融能力-消费关系的影响。本文通过构建金融能力综合指标，填补这一空白。


## 二、文献综述
### （一）金融能力的内涵及其影响因素
#### 1. 内涵界定
- 英国2006年调查：金融能力是“运用金融知识、实施理想金融行为以实现金融幸福”的能力，包括理财（收支平衡）、记录（支出透明）、计划（未来导向）、选品（合理决策）、明智（信息搜寻）5维度（Atkinson et al.，2006）；  
- 世界银行（2013）：在特定社会经济条件下，消费者做出最优金融决策的能力，核心是“知识+技能+态度+行为”；  
- 贝多广（2019）：个体在内外约束下做出合理判断的“金融知识+技能+生态+实践行为”，最终通过金融行为体现。

#### 2. 影响因素
- **内在因素**：教育背景（Atkinson，2007）、年龄（18-24岁群体金融能力最低，Xiao et al.，2015）、收入（农村居民收入越高，金融能力越强，李明贤和吴琦，2018）、金融教育经历（接受过金融教育者得分更高，Xiao and Neill，2016）；  
- **外在因素**：金融产品可得性、教育环境、经济金融环境、社会环境（Johnson and Sherraden，2007；Sherraden，2013）。


### （二）家庭消费的传统影响因素
经典理论（弗里德曼永久收入假说、莫迪利亚尼生命周期假说）指出，**永久性收入**与**财富**是消费的核心影响因素：  
- **收入层面**：消费对永久性收入冲击的反应强于暂时性冲击，且对预期收入下降不敏感（Jappelli and Pistaferri，2010）；收入增长对低消费人群的促进作用大于高消费人群（龙海明和钱浣秋，2018）；  
- **财富层面**：房产财富效应大于股票财富效应（Carroll et al.，2007）；地区财富分布冲击可解释美国2006-2009年消费下降（Atif et al.，2013）；我国城镇居民手持现金影响滞后、储蓄呈“挤占-促进”交替、证券资产有短期财富效应（罗娟和文琴，2016）。


### （三）金融能力对家庭消费的影响
现有研究聚焦金融知识、金融态度等单一维度，证实其对消费的正向作用：  
- 金融素养提升风险偏好，影响金融行为（朱涛等，2016）；客观与主观金融素养均与消费水平正相关，年轻群体中主观金融知识相关性更强（Henager and Cude，2016）；  
- 金融素养与消费增长正相关（Jappelli and Padula，2017）；金融知识影响信用卡消费，对高资产家庭作用最显著（张峰，2017）；  
- 作用机制包括：优化资产组合促进财富积累（吴雨等，2016）、提升正规信贷可得性（宋全云等，2017）、通过社会资本提高消费信贷可获得性（温雪，2018）、推动消费信贷使用与商业保险购买（宋全云等，2019）。  

但现有研究存在局限：未覆盖金融能力全维度，且忽略我国城乡与地区差异，本文将弥补这一缺陷。


## 三、金融能力指标的构建
### （一）指标构建依据
以**世界银行2013年金融能力框架**（金融知识、金融技能、金融态度、金融行为）为基础，结合Lusardi and Mitchell（2006）四原则：  
1. **简单化**：问题清晰，聚焦金融基本概念；  
2. **相关性**：与家庭日常金融决策关联；  
3. **简洁性**：指标独立、代表性强；  
4. **可区分性**：能区分不同金融素养水平。


### （二）指标体系构建
基于中国家庭金融调查（CHFS）数据，设计二级指标并赋值，具体如下表：

| 一级指标 | 二级指标       | CHFS 调查中的问卷题目                                                                 | 选项赋值                                                                 |
|----------|----------------|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 金融知识 | 利率           | 100元存5年定期（年利率4%），本息和是否大于120元？（1.小于；2.等于；3.大于；4.算不出） | 算不出=0；小于/大于=1；等于=2                                            |
|          | 通货膨胀       | 100元存1年（年利率5%，通胀3%），购买力变化？（1.增多；2.不变；3.减少；4.算不出）     | 算不出=0；不变/减少=1；增多=2                                            |
|          | 投资风险       | 单只股票风险是否大于股票基金？（1.是；2.否；3.没听过股票；4.没听过基金；5.都没听过） | 没听过任一项=0；否=1；是=2                                              |
| 金融技能 | 读写和计算技能 | 文化程度（1.没上学；2.小学；…；9.博士）                                               | 没上学=0；大专及以上=2；其他=1                                          |
|          | 理解运用能力   | 家庭投资股票时长（月）                                                               | 未开户=0；≤1年=1；>1年=2                                                |
| 金融态度 | 储蓄偏好       | 是否有人民币活期存款账户？（1.有；2.没有）                                           | 没有=0；有=1                                                             |
|          | 风险态度       | 投资偏好（1.高风险高回报；…；5.不愿承担风险）                                         | 不愿承担风险/不知道=0；其他=1                                            |
|          | 对未来的态度   | 未来5年是否计划新购/新建住房？（1.是；2.否；3.不清楚）                               | 是=1；否/不清楚=0                                                        |
| 金融行为 | 日常资金管理   | 家庭可支配收入是否覆盖消费性支出                                                     | 覆盖=1；未覆盖=0                                                         |
|          | 长期计划       | 是否有退休工资/社会养老保险？（1.公务员退休工资；…；10.都没有）                       | 都没有=0；其他=1                                                        |
|          | 选择金融产品   | 是否持有定期存款、股票、基金等金融资产                                               | 未持有=0；持有1种及以上=1                                                |

#### 信度检验
采用克朗巴哈Alpha系数检验内部一致性，测算结果为**0.8165**（处于0.8-0.9区间），表明指标体系可靠性良好。


### （三）综合指标衡量方法
借鉴杨丽和孙之淳（2015）的**熵值法**，计算每年一级指标的权重，进而合成金融能力综合指标（因金融能力具有稳定性，2015与2017年金融知识、风险态度指标一致）。


## 四、金融能力影响分类消费的实证分析
### （一）数据和样本
数据源自**中国家庭金融调查（CHFS）2013-2017年追踪数据**，覆盖全国29省355县（区、市），经筛选后保留：  
- 2015年：15583户（对2013年的追踪样本）；  
- 2017年：12084户（对2013年的追踪样本）。


### （二）模型设定和关键变量
#### 1. 模型设定
采用**面板数据双向固定效应模型**，考察金融能力对家庭分类消费的影响：  
\[Consump _{i t}=\beta_{0}+\beta_{1} * Finan\_capab _{i t}+\beta_{2} * X_{i t}+\theta_{i}+\mu_{t}+\varepsilon_{i t}\]  
- \(Consump_{it}\)：第i户第t年的消费水平（总消费/质量型消费）；  
- \(Finan\_capab_{it}\)：第i户户主第t年的金融能力；  
- \(X_{it}\)：家庭随时间变化的控制变量；  
- \(\theta_i\)：家庭固定效应（控制不随时间变化的因素）；  
- \(\mu_t\)：年份固定效应；  
- \(\varepsilon_{it}\)：随机扰动项（标准误聚集在省份层面）。

#### 关键概念界定
- **生存型消费**：食品、衣着、居住（水电燃料、物业费等）、日用品及服务；  
- **质量型消费**：休闲娱乐、旅游、通信交通、医疗保健、教育培训；  
- **信贷约束**：申请贷款被拒、因预期被拒未申请（经营性贷款）；需要但未申请、申请被拒（房产/汽车贷款），取值0-5（分数越高约束越强），样本中13%家庭存在信贷约束。

#### 4. 内生性检验（工具变量法）
选取“对经济金融信息的关注度（mess_foc）”作为工具变量（满足相关性：影响金融能力；外生性：不直接影响消费），采用两阶段回归：

| 变量         | 第一阶段（finan_capab） | 第二阶段（lntot_con） | 第二阶段（lnqua_con） |

## 六、结论与后续研究方向
### （一）核心结论
1. 居民金融能力对家庭总消费（尤其质量型消费）显著正向影响；  
2. 促进效应存在异质性：  
   - 城乡：城镇金融能力对质量型消费的作用>农村；  
   - 地区：西部金融能力对总消费与质量型消费的作用>东、中部；东部金融能力对质量型消费的作用>总消费；  
3. 作用机制：金融能力通过**提升正规负债（缓解信贷约束）** 、**增加家庭财富**、**扩大亲缘关系网络**，促进消费升级。


### （二）政策建议
1. **加强金融教育**：针对农村与西部地区，普及金融知识、提升金融技能，缩小城乡/地区金融能力差距；  
2. **优化信贷支持**：扩大正规信贷覆盖范围，缓解家庭信贷约束，尤其支持质量型消费需求；  
3. **重视社会资本**：通过社区建设、亲缘互助平台，强化家庭关系网络对消费的支撑作用。


### （三）后续研究方向
1. 深入分析地区家庭异质性（如收入水平、资产结构）对金融能力-消费关系的影响，提出更精准的区域性政策；  
2. 拓展数据维度，研究**网络消费**（2013-2017年CHFS数据显示人均网络消费增长15.23%）与金融能力的关联，适配数字经济下的消费新趋势。
